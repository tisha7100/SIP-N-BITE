# SIP-N-BITE 🍽️

SIP-N-BITE is a Laravel-based Restaurant Reservation and Food Ordering System.

## Features
- User Authentication
- Restaurant Reservation
- Food Ordering
- Cart Management
- Admin Dashboard
- Responsive UI

## Technologies Used
- Laravel
- PHP
- MySQL
- JavaScript
- Bootstrap

## Installation

```bash
git clone https://github.com/tisha7100/SIP-N-BITE.git
cd SIP-N-BITE
composer install
php artisan serve
```
